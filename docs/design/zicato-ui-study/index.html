harmonograf — a UI study in the zicato design language

theme

harmonograf is a multi-agent coordination console — a Gantt-forward instrument for watching many agents, 9 span kinds and goldfive's steering unfold over wall-clock. This study redraws it in zicato's design language: Tufte data-ink line-art, a calm mono ground, theme-adaptive by pure CSS swap. It is a design artifact — standalone HTML/CSS/JS, no build step, opens from file:// — theme-switchable across all 16 Console palettes.

The thesis — a full categorical palette, sourced from terminal ANSI

zicato spends exactly one green accent on a neutral ground — perfect for a binary promote/reject surface. harmonograf is a categorical instrument: it needs many distinguishable hues at once. So this study keeps the full categorical palette — but sources every categorical hue from the active theme's terminal ANSI palette (the Gogh scheme each zicato theme derives from). Switch the theme: the whole categorical encoding re-skins, ANSI and all.

That keeps harmonograf's elegant 2-axis encoding intact while honoring zicato's "earned by direction" rule. KIND is an identity hue; STATUS is a treatment. good / bad / accent appear only for status — so a span is never red for what it is, only when it failed.

KIND = hue (ANSI slot)

LLM_CALL — blue
TOOL_CALL — cyan
USER_MESSAGE — green
TRANSFER — yellow
WAIT_FOR_HUMAN — magenta

STATUS = treatment (role)

RUNNING — accent + breathe (now)
✕ FAILED — bad + glyph
COMPLETED ok — good
PENDING — faint (neutral, not red)
CANCELLED — hatch

The surfaces

shell →
The chrome: AppBar → console topbar (the harmonograf α mark + plain-letter wordmark, breadcrumbs, status / RUN pill), NavRail, CurrentTaskStrip, TransportBar, inspector Drawer. Before/after per element.
gantt →
The hero. The execution timeline in line-art: categorical kind hues, status treatments, the accent now-cursor, agent lifelines, context band. Both the SVG and canvas options.
sequence →
How the agents talk to one another. The agent-interaction sequence diagram (lifelines down, time down, each message a directional connector) paired with the directional transfer chord for topology; the horizontal score is the Gantt-companion alternative. Lifelines stay neutral; messages carry the kind hue; direction is explicit.
figures →
The component catalogue: OrchestrationTimeline, TaskStages, LiveActivity (.dn-board-table), Interventions (goldfive steering), the ⌘K SessionPicker.
brand →
harmonograf's own mark. A logo study for the standalone console — the double namesake (Adamiecki's 1896 harmonogram × the Victorian harmonograph), four line-art marks generated live from the damped-oscillator equations, wordmark, favicon cuts, the accent question.
grammar →
Fourteen new figures. Drift seismograph, intervention ladder, correction slopegraphs, drift sankey, plan strata, replan slopegraph, plan DAG (critical path as the spine), delegation score, transfer chord, conductor's marks, context tide, the harmonograph session fingerprint.
compose →
The full console, interactive. Three whole-UI proposals — A·observatory (rail-switched hero views + inspector), B·score (sequence-first with a gantt minimap and drift margin), C·mission control (fingerprint fleet) — with clickable spans, ⌘K session switching, and the grammar figures composed in. Opens from file://, no server needed.
