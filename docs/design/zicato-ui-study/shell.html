accent reserved · now / live / selected only theme

harmonograf's chrome today is Material Design 3: --md-sys-color-* tokens, a system-ui sans, elevation containers with drop shadows, a blue primary, and several animation:…infinite pulses. This study redraws every chrome surface in zicato's language — one calm --paper ground with a single --panel lift, hairline --rule separators instead of elevation, an all-mono voice, and --accent reserved for now / live / selected. Switch the theme above: the whole shell re-skins across all 16 Console palettes with no re-render.

AppBar → topbar

The MD3 AppBar is a flat container with a sans title and a surface-tinted session button. The console topbar leads with the harmonograf lockup — the mirrored-Lissajous α mark beside plain-letter harmonograf (one alpha per lockup; see the brand study), then mono breadcrumbs, the typeface switch and scale pill, and a status pill whose RUN badge pulses the one accent — the chrome's only keyframe (gated under prefers-reduced-motion).

NavRail & CurrentTaskStrip

MD3 marks the selected rail item with a secondary-container fill (green pill) and animates the running task chip with an infinite opacity pulse. In-language: the selected item is a quiet color-mix(--accent 14%, --panel) tint with accent text; the strip's running state is a mono outline-first accent pill — not red (an in-flight task is neutral, never a failure).

TransportBar

Transport controls become mono outline-first buttons (fill the accent only on hover). The LIVE badge keeps a single pulsing dot — re-pointed from the MD3 blue primary to --accent — and the clock uses tabular-nums so digits never jitter. The "return to live" affordance, an MD3 amber container, maps to a --caution outline.

Inspector Drawer

The drawer keeps its tabbed shape but loses the elevated surface-container-highest code block in favor of a hairline-framed --panel-2 pre. Tabs underline in --accent when active; the JSON tint reads from tokens (--good strings, --caution numbers, --bad booleans, --accent keys) so it re-skins with the theme instead of shipping fixed syntax colors.
